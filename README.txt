注意，在constraints文件里面输入构造的约束参数，有以下几点注意事项：
1.生成的q为3n维，n为输入的i，j的类别个数，排列方式是[x0,y0,phi0,x1,y1,phi1....]，注意这里i，j最小的从0开始按顺序标！
2.驱动按照正常表达式，囊括所有基本函数。注意这里的自变量是t，乘法别忽略乘号否则识别不出来，
幂函数用^，尽量多用括号控制计算顺序（式子很复杂情况下不加括号不保证计算顺序），
规定驱动后请手动算一下一阶二阶导数填入c'与c''中（很简单的），python做不到自动算呜呜呜
3.初始默认q的所有位形都是零，如果有多个根不符合要求，请自行调整q0（很简单的）
4.表格输入的s的坐标中间用空格隔开，两端没有空格
5.由于Excel的规则，要输入如-0.5 0这样的向量时，要在前面加上英文的单引号，如'-0.5 0
6.注意保证约束个数（包括运动学约束与驱动约束）为构建个数的三倍以保证自由度数等于驱动个数

目前只开发了ax，ay，r三种运动学约束与aphid驱动，不久后更新

然后，在main.py中选择一下画什么后运行该主程序即可，如果初始化的q不好自己改改
