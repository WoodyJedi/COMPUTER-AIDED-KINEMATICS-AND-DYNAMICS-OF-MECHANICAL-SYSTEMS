注意，在constraints文件里面输入构造的约束参数，有以下几点注意事项：
1.生成的q为3*n维，n为输入的i，j的类别个数，排列方式是[x0,y0,phi0,x1,y1,phi1....]，注意这里i，j最小的从0开始按顺序标！
2.驱动按照正常表达式，囊括所有基本函数。注意这里的自变量是t，乘法别忽略否则识别不出来（不支持省略乘号的写法），
幂函数用^，尽量多用括号控制计算顺序（式子很复杂情况下不加括号不保证计算顺序），
规定驱动后请手动算一下一阶二阶导数填入c'与c''中（很简单的）
3.表格输入的s的坐标中间用空格隔开，两端没有空格
4.几何长度都要用实数形式表示出来（如果有L，d等变量都要代掉，单位要求cm）
5.由于Excel的规则，要输入如-0.5 0这样的向量时，要在前面加上英文的单引号，如'-0.5 0
6.注意保证约束个数（包括运动学约束与驱动约束）为构建个数的三倍以保证自由度数等于驱动个数（不检查）


如果有画动画的必要，在geometrical_parameters.xlsx中输入所有刚体的参数
1.动画需要输入刚体的几何参数，每一行代表一个刚体，两个端点的坐标目前只支持平面矩形，连体基的x轴为矩形沿着长边的中轴线
2.如有必要手动调整动画显示的坐标轴范围，进入show_animation.py找到    ax = fig.add_subplot(111, autoscale_on=False, xlim=(-10,10), ylim=(-10, 10))
这一行调整
3.如果要把动画导出，进入show_animation.py将# ani.save("test.gif",writer='pillow')的注释去掉


目前只开发了ax，ay，r, rt, t五种运动学约束与aphid驱动，不久后更新，如果用了处这几种之外的约束会，导致程序出错，打印出警告

然后，在main.py中选择一下画哪些曲线图后运行该主程序即可，如果初始化的q不好自己改改
